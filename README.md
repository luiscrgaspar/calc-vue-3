# calc-vue-3

Simple scientific calculator built with Vue 3, TypeScript, Vuex, and vue-i18n.

## Setup

```bash
yarn install
```

## Development

```bash
yarn start
```

## Build

```bash
yarn build
```

## Tests

```bash
yarn test
```

## Lint

```bash
yarn lint
```
